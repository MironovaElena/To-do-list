1.Из скольких частей состоит основной цикл? Цикл состоит из условия и тела. 2.Какое количество уровней вложенности могут иметь циклы? Циклы могут иметь любое количество уровней вложенности. 3.Если switch находится внутри функции, какая инструкция может использоваться для выхода из него? Для выхода из инструкции switch может использоваться не только break, но также и инструкции return или continue, если switch находится внутри 4.Можно ли пропускать части for? Что получится, если написать for(;;)? Любая часть for может быть пропущена. Если написать for(;;) то будет бесконечный цикл. 5.Самостоятельно разберитесь, как работает цикл while и приведите два примера кода с его использованием. Цикл while (“пока”) позволяет выполнить одну и ту же последовательность действий, пока проверяемое условие истинно. Условие записывается до тела цикла и проверяется до выполнения тела цикла. Как правило, цикл while используется, когда невозможно определить точное значение количества проходов исполнения цикла. let i = 0; while (i < 3) { // выводит 0, затем 1, затем 2 alert( i ); i++; } 6.Как можно принудительно остановить выполнение цикла? Поставить после if оператор break. Оператор break завершает действие в котором он находится. 7.При помощи цикла for выведите чётные числа от 2 до 10. for (let i = 0; i < 10; i++) { if (i % 2) { console.log( i ); } } 8.Если тело цикла состоит лишь из одной инструкции, мы можем опустить фигурные скобки {…}? Если цикл содержит одну инструкцию, то фигурные скобки можно опустить: int i {}; while(++i < 10) 9.Что выведет цикл? for (let i = 0; i < 3; i++) { console.log(i); } выведет: 0 1 2 10.Оба цикла выводят в консоль одинаковые значения или нет? let i = 0; while (++i < 5) console.log(i); // и let i = 0; while (i++ < 5) console.log(i); нет 1. выведет 1 2 3 4 а 2. выведет 1 2 3 4 5
